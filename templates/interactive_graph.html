Интерактивное построение графа

Показать/скрыть инструкции
Инструкция по использованию:
  • Добавить вершину: Нажмите кнопку "Добавить вершину", затем кликните на поле графа, чтобы добавить вершину
  • Добавить ребро: Нажмите кнопку "Добавить ребро", затем выберите первую вершину, затем вторую, чтобы добавить ребро между этими вершинами
  • Удалить вершину: Нажмите кнопку "Удалить вершину", затем кликните на вершину, чтобы удалить ее
  • Удалить ребро: Нажмите кнопку "Удалить ребро", затем кликните на ребро, чтобы удалить его
  • Переименовать вершину: Нажмите "Переименовать вершину", затем кликните на вершину, чтобы поменять ее имя
  • Изменить цвет вершины: Нажмите "Изменить цвет вершины", затем кликните на вершину, чтобы изменить ее цвет
  • Изменить цвет ребра: Нажмите "Изменить цвет ребра", затем кликните на ребро, чтобы поменять его цвет
  • Изменить вес: Введите новый вес, нажмите "Изменить вес", затем кликните на ребро, чтобы добавить ему вес
  • Удалить вес: Нажмите "Удалить вес", затем кликните на ребро, чтобы удалить его вес
  • Тип графа: Выберите "Ориентированный" или "Неориентированный"
  • Очистить граф: Кнопка "Очистить граф" удалит все элементы
  • Сохранить как: Сохраняет граф в формате PNG/SVG/JSON/TXT
  • Алгоритмы: Выберите "Матрицы", чтобы построить матрицу смежности/инцидентности/... графа
  • Вставить граф: Вставить граф в формате JSON/TXT
  • Выровнять граф: Нажмите кнопку "Выровнять граф", чтобы автоматически выровнять элементы графа
