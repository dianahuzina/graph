**Требования**: Python 3.8 или новее


**Как запустить приложение:**


  **1. Скачать репозиторий**
  
      Нажать зелёную кнопку Code -> Download ZIP и распаковать архив.
          
      Или через терминал:
          git clone https://github.com/dianahuzina/graph.git
          cd graph

          
  **2. Установить зависимости**
  
    Открыть терминал в папке с проектом и выполнить:
        pip install flask networkx numpy matplotlib
        
  **3. Запустить приложение**
  
        python App.py
        
  **4. Открыть в браузере**
  
        Перейти по адресу: http://127.0.0.1:5000
