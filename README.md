# Веб-приложение для работы с графами

Интерактивное веб-приложение для построения и визуализации графов и мультиграфов. Проект позволяет вычислять и отображать различные характеристики графов, включая матрицы смежности, инцидентности, путей и Кирхгофа, а также находить кратчайшие пути с помощью алгоритма Флойда-Уоршелла.

## Требования
* Python 3.8 или новее

## Как запустить приложение на локальном компьютере

### 1. Скачать проект
Склонируйте репозиторий через терминал:
```bash
git clone [https://github.com/dianahuzina/graph.git](https://github.com/dianahuzina/graph.git)
cd graph
```

Либо нажмите зелёную кнопку Code -> Download ZIP на GitHub, распакуйте архив и откройте эту папку в терминале

### 2. Установить зависимости
Убедитесь, что вы находитесь в папке проекта, и выполните команду:
```bash
pip install flask networkx numpy matplotlib
```

### 3. Запустить приложение
Запустите главный файл сервера:
```bash
python App.py
```

### 4. Открыть в браузере
После успешного запуска сервера перейдите по локальному адресу: http://127.0.0.1:5000
